Rule-Based AI Chatbot

Overview

This project is a Rule-Based AI Chatbot developed to simulate human-like conversations using predefined rules and responses. The chatbot identifies user inputs and provides appropriate replies based on a set of programmed conditions.

Features

- Interactive text-based conversation
- Predefined question and answer patterns
- User-friendly interface
- Fast response generation
- Easy to customize and expand

Technologies Used

- Python
- Rule-Based Logic
- String Matching Techniques

Project Structure

Rule-Based-AI-Chatbot/
│
├── chatbot.py
├── responses.py
├── requirements.txt
└── README.md

How It Works

1. The user enters a message.
2. The chatbot analyzes the input.
3. The system matches the input with predefined rules.
4. The chatbot returns the corresponding response.

Installation

1. Clone the repository:

git clone https://github.com/yourusername/Rule-Based-AI-Chatbot.git

2. Navigate to the project directory:

cd Rule-Based-AI-Chatbot

3. Run the chatbot:

python chatbot.py
